 Автоматизация закупок и оптимизация запасов (Supply Chain Analytics)
 
📌 О проекте:

Разработка системы поддержки принятия решений (DSS) для отдела закупок. Проект переводит ручной процесс формирования заказов из сложного Excel-отчета в автоматизированный алгоритм на Python, минимизируя «человеческий фактор» и оптимизируя оборотный капитал компании.

🛠 Технический стек:

Python (Pandas, NumPy): Обработка «грязных» данных (ETL), нормализация заголовков, обработка объединенных ячеек (ffill).
Модель прогнозирования: Взвешенное скользящее среднее (Weighted Moving Average — WMA) с коэффициентами 0.5 (тек. месяц) / 0.3 / 0.2.
Бизнес-логика: Учет страховых запасов (нормативов), товаров в пути (In-Transit) и кратности логистических единиц (паллеты/короба).

🚀 Ключевые функции алгоритма:

Data Cleaning: Автоматическое восстановление данных из сложных Excel-структур и очистка от невидимых символов.
Smart Forecasting: Алгоритм выявляет рыночные тренды за 90 дней, предотвращая Overstock (затоваривание) при случайных всплесках спроса.
Auto-Ordering: Автоматический расчет потребности и округление до кратности упаковки (аналог логики ОКРВВЕРХ).
Export: Генерация готового файла заказа для поставщика в формате .xlsx.

📊 Результаты:

Экономический эффект: На примере поставщика «Хаят Маркетинг» алгоритм позволил сократить объем избыточного заказа на 17881 ед. товара и экономия места на складе (ориентировочно): 28.2 паллет по сравнению с линейным расчетом.
Эффективность: Сокращено время расчета заказа для всей номенклатуры.
